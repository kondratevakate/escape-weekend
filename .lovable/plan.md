

# План: Улучшение UI кнопок лайка в карточке места

## Проблема

1. **Подсветка рамки** при свайпе (зелёная при лайке) выглядит странно и непонятно
2. **Кнопка Like внутри карточки** маленькая и теряется
3. **Круглые кнопки снизу** (X и сердечко) дублируют функционал и плохо видны

## Предлагаемое решение

Переместить кнопку лайка на фото места (как в референсе Wanderlog — сердечко в правом верхнем углу фото). Убрать дублирующие элементы.

### Новый UI

```text
┌─────────────────────────────────────┐
│  [Фото места]         ❤️ 5    [X]  │  <- Лайк на фото, справа
│  ★ 4.5 (151)                        │
├─────────────────────────────────────┤
│  🏔️ Природа                         │
│  Сейдозеро                          │
│  ❤️ 246  📤 52                       │
├─────────────────────────────────────┤
│  ✨ AI summary                      │
│  Visitors praise the incredible...  │
├─────────────────────────────────────┤
│  📝 Отзывы (3)                 [▼]  │
├─────────────────────────────────────┤
│          [📤 Share]                  │  <- Только Share кнопка
└─────────────────────────────────────┘
```

## Что изменится

### PlaceCard.tsx

1. **Убрать swipe indicators** (рамка подсветки) — строки 128-136
2. **Переместить кнопку лайка на фото** — рядом с крестиком закрытия
   - Показывать счётчик лайков рядом (как на скриншоте — "❤️ 5")
   - При нажатии — анимация заполнения сердечка
3. **Убрать кнопку Like из Actions** — оставить только Share
4. **Убрать круглые кнопки внизу** (строки 319-338) — они дублируют функционал

### Визуальный стиль кнопки лайка

- Полупрозрачный фон (как у кнопки закрытия)
- Сердечко + счётчик
- При наведении — затемнение фона
- При лайке — заполненное красное сердечко
