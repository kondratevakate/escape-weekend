
# План: Добавить слой истории — территории и культура саамов

## Что добавляем

На карту Кольского полуострова добавится переключаемый "Слой истории" с:
1. **Полигон** — историческая территория расселения саамов (покроет Кольский полуостров полупрозрачной заливкой)
2. **Новые точки интереса** — саамские музеи, поселения и культурные объекты
3. **Переключатель слоя** — кнопка для вкл/выкл исторического слоя

## Данные для слоя

### Полигон территории саамов (GeoJSON)
Координаты охватывают Кольский полуостров — основную территорию проживания саамов в России.

### Новые точки интереса (категория "history")

| Место | Координаты | Описание |
|-------|------------|----------|
| Ловозеро | 68.0°, 35.0° | Центр саамской культуры в России |
| Музей истории саамского народа | 68.0°, 35.0° | Этнографический музей |
| Сейдозеро (священное) | 67.82°, 34.85° | Священное озеро саамов |
| Оленегорск (исторический) | 68.15°, 33.28° | Исторический центр оленеводства |
| Краеведческий музей Ревды | 68.05°, 34.55° | Музей с саамской экспозицией |
| Саам-Суэль (село Териберка) | 69.16°, 35.12° | Древнее саамское поселение |

## Техническая реализация

### 1. Новая категория в kolaPlaces.ts

```typescript
export type PlaceCategory = 
  | 'nature' 
  | 'hiking' 
  | 'museum' 
  | 'attraction' 
  | 'village' 
  | 'city' 
  | 'reserve'
  | 'history';  // ← НОВОЕ

// В categoryConfig добавить:
history: {
  label: 'История',
  icon: '📜',
  color: 'hsl(340 65% 45%)',    // бордовый
  bgColor: 'hsl(340 65% 95%)',
},
```

### 2. GeoJSON полигон территории

```typescript
// src/data/saamiHistoryLayer.ts
export const saamiTerritoryGeoJSON = {
  type: 'Feature',
  properties: { name: 'Территория саамов' },
  geometry: {
    type: 'Polygon',
    coordinates: [[
      [28.0, 69.5],   // СЗ угол
      [41.0, 69.5],   // СВ угол  
      [41.0, 66.0],   // ЮВ угол
      [28.0, 66.0],   // ЮЗ угол
      [28.0, 69.5],   // замыкаем
    ]],
  },
};
```

### 3. Отрисовка в MapView

```typescript
// В useEffect инициализации карты:
const historyLayerRef = useRef<L.GeoJSON | null>(null);

// При включении слоя истории:
if (showHistoryLayer && !historyLayerRef.current) {
  historyLayerRef.current = L.geoJSON(saamiTerritoryGeoJSON, {
    style: {
      fillColor: 'hsl(340, 65%, 45%)',
      fillOpacity: 0.15,
      color: 'hsl(340, 65%, 35%)',
      weight: 2,
      dashArray: '5, 5',
    }
  }).addTo(mapInstanceRef.current);
}
```

### 4. UI переключателя

В `CategoryFilter.tsx` добавить кнопку:

```tsx
<button
  onClick={onToggleHistoryLayer}
  className={showHistoryLayer ? 'bg-history/20' : ''}
>
  <span>📜</span>
  <span>История</span>
</button>
```

## Источник данных

Информация о саамах взята из "Интерактивного атласа коренных малочисленных народов" (atlaskmns.ru):
- Саамы — коренной народ Кольского полуострова (около 1991 чел. в РФ)
- Основные центры расселения: Ловозеро, Ревда
- Традиционные занятия: оленеводство, рыболовство
