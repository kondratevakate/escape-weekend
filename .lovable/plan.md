
# План: Убрать дублирование лайков + добавить Double-tap

## Проблема

Счётчик лайков показывается **дважды**:
1. На фото (кнопка с сердечком + число)
2. В контенте (секция "Social stats" с ❤️ и 📤)

## Самые популярные паттерны мобильных лайков

| Приложение | Паттерн | Описание |
|------------|---------|----------|
| **Instagram** | Double-tap на фото | Двойной тап → большое сердце по центру фото |
| **TikTok** | Double-tap + кнопка | Одна кнопка справа + double-tap на видео |
| **Tinder** | Swipe right | Свайп вправо = лайк (уже есть у вас) |

## Рекомендация

Использовать **комбинацию Instagram + Tinder**:
- **Double-tap на фото** — интуитивно для мобильных пользователей
- **Свайп вправо** — уже работает
- **Убрать дублирующую статистику** — оставить только на фото

### Анимация double-tap

При двойном тапе на фото:
1. Большое сердце (40x40) появляется по центру фото
2. Scale-анимация: 0 → 1.2 → 1 → исчезает
3. Добавляется лайк (если ещё не лайкнуто)

## Техническая реализация

```tsx
// Double-tap логика
const lastTap = useRef(0);
const DOUBLE_TAP_DELAY = 300;

const handleImageTap = () => {
  const now = Date.now();
  if (now - lastTap.current < DOUBLE_TAP_DELAY) {
    // Double tap!
    setShowBigHeart(true);
    if (!isFavorite) onToggleFavorite();
    setTimeout(() => setShowBigHeart(false), 800);
  }
  lastTap.current = now;
};

// Большое сердце по центру фото
{showBigHeart && (
  <Heart className="absolute top-1/2 left-1/2 -translate-x-1/2 -translate-y-1/2 
                    h-16 w-16 fill-red-500 text-red-500 
                    animate-heart-pop pointer-events-none" />
)}
```
