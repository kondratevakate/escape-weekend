## Блок 3. Персоны пользователей

Создать `docs/personas.md` — 5 персон на базе типичных профилей vas3k.club, адаптированных под Escape Weekend.

**Структура каждой персоны:**
- Имя + одна строка
- Возраст, профессия, город
- Что любит / как путешествует
- Главная боль
- Какие фичи Escape Weekend закрывают эту боль
- Цитата

**Список персон:**
1. **Артём, 32, продакт-менеджер из Москвы** — выгорел, едет на выходные «куда угодно подальше». Хочет сюрприз-маршрут, не любит планировать.
2. **Маша, 28, фронтенд-разработчица из Питера** — фотограф-любитель, ездит на сияние и киты. Ищет нишевых экспертов и проверенные точки.
3. **Денис и Аня, 35/33, пара из Екатеринбурга** — отпуск 2 раза в год, бюджет средний, дети-нет. Хотят «чтобы было душевно и без туристов».
4. **Игорь, 41, основатель стартапа из Казани** — соло-путешествия, активный отдых (хайкинг, каяк). Ценит честные предупреждения об опасностях.
5. **Лена, 26, дизайнер-фрилансер, номад** — живёт между городами, ищет рестораны/локальную кухню/места «не для всех». Активный пользователь Telegram.

Каждая персона будет связана с конкретными фичами проекта: Stash, Trip Planner, Клуб, слой ресторанов, hazards, Лаборатория идей.
