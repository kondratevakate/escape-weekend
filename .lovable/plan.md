
## Цель
Сделать русский язык дефолтным во всём приложении и проверить ключевые флоу end-to-end в браузере.

## Что меняем

### 1. Дефолтный язык — RU для всех
Файл `src/contexts/LanguageContext.tsx`:
- Убрать определение языка из `navigator.language` — всегда возвращать `'ru'`, если в localStorage пусто.
- Сохранить возможность переключиться на EN через `LanguageSwitcher` (английский остаётся как опция, не удаляем).

```ts
const getInitialLanguage = (): Language => {
  try {
    const stored = localStorage.getItem(STORAGE_KEY) as Language | null;
    if (stored === 'ru' || stored === 'en') return stored;
  } catch {}
  return 'ru'; // было: detect from browser
};
```

### 2. Аудит хардкоженных английских строк
Пройтись по компонентам и найти строки, которые написаны только по-английски (без `t()` или `language === 'ru' ? ... : ...`). Кандидаты по результатам беглого осмотра:
- `src/components/map/AddMapModal.tsx` — toast'ы и плейсхолдеры выглядят ок (двуязычные), проверить.
- Компоненты `src/components/landing/*` — Hero, Footer, ResourcesSection — проверить, что все строки локализованы.
- `src/components/club/*` — PostCard, CommentThread, ReactionButton.
- `src/pages/Club*.tsx` — уже двуязычные, проверить полноту.
- `src/components/CookieConsent.tsx`, `WelcomeModal.tsx`, `LockedScreen.tsx`, `PremiumGate.tsx`.

Где находим английский без перевода — добавляем условие `language === 'ru' ? '…' : '…'` или ключ в `i18n.ts`.

### 3. HTML lang
`index.html` — поменять `<html lang="en">` на `<html lang="ru">` (если стоит en).

### 4. End-to-end проверка в браузере
После правок прогоняем через browser-инструменты основные сценарии:

1. **Главная `/`** — RU по умолчанию, нет английских хвостов в Header / Hero / Discover / Footer.
2. **Карта** — фильтры, легенда, переключатели слоёв (UNESCO, Indigenous, Restaurants, Hazards) — всё на русском.
3. **Клик по месту** → PlaceCard — секции «Когда», «Как добраться», «Только здесь», «Из клуба».
4. **`/club`** — лента, фильтры по нише и типу, сортировки.
5. **`/club/join`** — форма заявки, валидация, кнопка «Отправить через Telegram».
6. **`/stash`** — пустое состояние и с сохранёнными местами.
7. **Переключатель языка** — переключение RU↔EN работает, выбор сохраняется в localStorage.
8. **Mobile (375x667)** — нижняя навигация, мобильное меню.

Если по ходу теста находим баги или непереведённые строки — фиксим точечно.
