
# Комплексный план развития: Исследуй Кольский

## Обзор новых функций

Добавляем 6 крупных функций:
1. Избранное со свайп-механикой (Tinder-стиль) + режим Explore
2. Маршруты с визуализацией транспорта (поезд, самолёт, малая авиация, по морю)
3. Переключение языка (RU/EN)
4. AI-сайдбар для помощи и рекомендаций
5. Cookie Policy
6. Авторизация + лайки мест (требует Supabase)

---

## Часть 1: Избранное со свайпами

### 1.1 Хранилище избранного
**Файл:** `src/hooks/useFavorites.ts`

Хук для работы с localStorage:
- `favorites: string[]` — массив ID мест
- `toggleFavorite(id)`, `isFavorite(id)`
- Синхронизация с localStorage
- После подключения Supabase — синхронизация с БД

### 1.2 Свайп-жесты на карточке места
**Файл:** `src/components/map/PlaceCard.tsx`

- Свайп вправо → добавить в избранное (зелёная подсветка)
- Свайп влево → закрыть (красная подсветка)
- Кнопки ❌ и ❤️ под карточкой
- CSS-анимация "улетания"

### 1.3 Режим Explore (полноэкранный Tinder)
**Файл:** `src/components/map/ExploreMode.tsx`

- Полноэкранные карточки одна за другой
- Свайп или кнопки для действий
- Счётчик "5 из 24 мест"
- Фильтрация непросмотренных
- Кнопка выхода на карту

### 1.4 Выделение избранного на карте
**Файл:** `src/components/map/MapView.tsx`

- Золотая обводка + пульсирующее сердечко для избранных
- Визуальное отличие от обычных маркеров

### 1.5 Фильтр "Только избранное"
**Файл:** `src/components/map/CategoryFilter.tsx`

- Специальная кнопка ❤️ "Избранное"
- Работает в комбинации с категориями

### 2.4 Управление слоями

Панель справа с чекбоксами:

## Технические детали

### Свайп-жесты
Реализация через touch events без дополнительных библиотек:
- `onTouchStart` — запомнить начальную позицию
- `onTouchMove` — отслеживать смещение, применять CSS transform
- `onTouchEnd` — определить направление, выполнить действие

### Leaflet маршруты
- `L.polyline()` для линий
- `L.divIcon()` для иконок транспорта
- `L.layerGroup()` для группировки

### Хранение языка
- localStorage ключ `app_language`
- Дефолт определяется по `navigator.language`

### Авторизация
- Supabase Auth с email/password
- Опционально: OAuth (Google, GitHub)
- JWT токены для API
