

# Редизайн лендинга в стиле Airbnb

## Общая концепция

Превращаем сайт из "лендинг + карта" в "сразу карта с карточками" — как Airbnb показывает жильё. Минималистично, фокус на контенте.

## Часть 1: Новый Header

**Файл:** `src/components/landing/Header.tsx`

```text
┌────────────────────────────────────────────────────────────┐
│ 🌌 Kola     [🏔️ Природа] [🥾 Походы] [⭐ Топ]       🌐  ≡ │
└────────────────────────────────────────────────────────────┘
```

Элементы:
- **Слева:** Логотип (эмодзи + название)
- **По центру:** 3 фильтра-кнопки с эмодзи (как табы Airbnb)
- **Справа:** 
  - 🌐 Глобус — переключение языка (dropdown RU/EN)
  - ≡ Бургер-меню (опционально)

### Фильтры по центру

3 упрощённых категории:
| Эмодзи | Название | Включает |
|--------|----------|----------|
| 🏔️ | Природа | nature, reserve |
| 🥾 | Походы | hiking |
| ⭐ | Топ-места | attraction, museum, village, city |

Или показывать все категории горизонтальным скроллом (как Airbnb Categories).

## Порядок реализации

1. **LanguageSwitcher** — добавить вариант с глобусом
2. **Header** — редизайн с фильтрами по центру
3. **PlacesList + PlaceListCard** — карточки слева
4. **Index** — split layout (карточки слева, карта справа)
5. **KolaMap** — упростить (убрать встроенный header)
6. **Мобильная адаптация** — drawer для карточек

```css
  display: flex;
  height: 100vh;
}
.cards-panel {
  width: 400px;
  overflow-y: auto;
```

### Синхронизация карта ↔ список
- При клике на карточку — подсветить маркер на карте
- При клике на маркер — прокрутить к карточке

### Упрощённые категории для фильтров

```typescript

};
```
