

# План: Эмодзи народов на карте + Личный кабинет Tourist/Creator

## Часть 1: Эмодзи коренных народов как маркеры на карте

### Текущая ситуация
Сейчас народы отображаются только как полигоны территорий. Культурные центры — это обычные маркеры с иконкой 📜.

### Решение
Добавим уникальные эмодзи для каждого народа, которые будут отображаться на карте как маркеры культурных центров.

### Выбор эмодзи

| Народ | Эмодзи | Почему |
|-------|--------|--------|
| Саамы | 🦌 | Оленеводство — основа культуры |
| Ненцы | 🏕️ | Чумы — традиционное жилище |
| Ханты | 🐻 | Культ медведя |
| Манси | 🌲 | Лесные охотники Урала |
| Эвенки | 🫎 | Лосиная охота |
| Эвены | 🏔️ | Горные оленеводы |
| Чукчи | 🐋 | Морской промысел |
| Коряки | 🌋 | Камчатские вулканы |
| Долганы | ❄️ | Полярные кочевники |
| Нанайцы | 🐟 | Рыболовы Амура |
| Селькупы | 🛶 | Речные охотники |
| Юкагиры | 🦅 | Древние охотники |

### Различия между ролями

#### Турист (Tourist)

**Левая панель "Источники":**
- Сохранённые места (❤️)
- Мои списки (📋)
- Загруженные гиды (📚)
- Скачанные слои (🗺️)

**Центральная панель "Чат":**
- Построй маршрут на N дней
- Что посмотреть рядом с...
- Оптимизируй мой маршрут
- Где остановиться около...

**Правая панель "Результат":**
- Готовый маршрут с картой
- Рекомендации по местам
- Апсейлы (платные гиды, экскурсии)
- Экспорт в PDF/GPX

#### Креатор (Creator)

**Левая панель "Источники":**
- Мои слои (🗺️)
- Мои коллекции (📦)
- Черновики (✏️)
- Опубликованные (✅)

**Центральная панель "Редактор":**
- Добавление мест на карту
- Описания и фото
- Категории и теги
- Превью слоя

**Правая панель "Статистика":**
- Просмотры слоя
- Сохранения
- Отзывы
- Доход (если монетизация)

### Структура данных

```typescript
// Роль пользователя
type UserRole = 'tourist' | 'creator';

// Расширенный пользователь
interface User {
  email: string;
  role: UserRole;
  displayName?: string;
  avatar?: string;
}

// Слой от креатора (для прототипа — моки)
interface UserLayer {
  id: string;
  authorId: string;
  name: string;
  description: string;
  places: Place[];
  isPublished: boolean;
  createdAt: Date;
}

// Маршрут туриста
interface Route {
  id: string;
  userId: string;
  name: string;
  days: RouteDay[];
  totalDistance: number;
}

interface RouteDay {
  date: string;
  places: Place[];
  notes: string;
}
```

## Порядок реализации

### Этап 1: Эмодзи народов на карте
1. Добавить `emoji` в `indigenousPeoplesLayer.ts`
2. Обновить рендеринг маркеров в `MapView.tsx`
3. Протестировать отображение

### Этап 2: Роли и онбординг
4. Расширить `AuthContext` — добавить `role`
5. Создать страницу `Onboarding.tsx`
6. Обновить `LoginModal` — после логина показывать выбор роли
7. Добавить роуты в `App.tsx`

### Этап 3: Личный кабинет туриста
8. Создать `Dashboard.tsx` с трёхколоночным лейаутом
9. `SourcesPanel` — списки мест
10. `ChatPanel` — мок чата с AI
11. `ResultPanel` — отображение маршрута

### Этап 4: Личный кабинет креатора
12. `CreatorDashboard` — редактор слоёв
13. Форма добавления мест
14. Превью слоя на карте
15. Публикация в общий каталог (мок)

## План проверки

| Шаг | Что проверяем | Ожидаемый результат |
|-----|---------------|---------------------|
| 1 | Включить слой народов на карте | Культурные центры показывают эмодзи народов (🦌, 🏕️, 🐋...) |
| 2 | Залогиниться | Появляется экран выбора роли |
| 3 | Выбрать "Турист" | Открывается личный кабинет с чатом |
| 4 | Написать в чат "Построй маршрут" | Появляется мок-ответ с маршрутом справа |
| 5 | Выбрать "Креатор" | Открывается редактор слоёв |
| 6 | Добавить место в свой слой | Место появляется на превью |
| 7 | Проверить на мобильном | Панели сворачиваются в табы/drawer |
