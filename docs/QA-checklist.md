# Инструкции для тестировщика — Hedonist Odyssey / WoWAtlas

Документ для ручного QA перед релизом. Прогоняй полный чек-лист на двух вьюпортах: **мобилка 375×812** и **десктоп 1366×768**. Где поведение различается — отмечено явно.

> Если нашёл баг — заведи issue по шаблону `.github/ISSUE_TEMPLATE/bug_report.md` и приложи скриншот + console-лог.

---

## 0. Подготовка

1. Открой превью или прод (`https://wowatlas.lovable.app`).
2. Перед каждым большим разделом чисти состояние: DevTools → Application → **Clear site data** (localStorage + cookies).
3. Открой две вкладки DevTools: **Console** и **Network** (фильтр JS).
4. Убедись, что язык по умолчанию — **русский** (без выставленного `wowatlas_lang` в localStorage).

## 1. Smoke-тест (5 минут)

| # | Шаг | Ожидаемый результат |
|---|-----|---------------------|
| 1.1 | Открыть `/` | Карта загружается ≤ 3 с, маркеры видны, ошибок в console нет |
| 1.2 | Открыть `/club` | Лента постов открывается, нет 404 |
| 1.3 | Открыть `/stash` | Пустое состояние с текстом-пояснением |
| 1.4 | Открыть `/trip-planner` | Страница открывается, нет белого экрана |
| 1.5 | Перейти на несуществующий `/foo` | 404-страница, ссылка «На главную» работает |

❌ **Стоп-баг**: любая красная ошибка в console, белый экран, бесконечный спиннер > 10 с.

## 2. Локализация (RU по умолчанию)

1. Очисти localStorage, перезагрузи `/`.
2. Проверь, что **весь интерфейс на русском**: header, sidebar, фильтры, footer, welcome-модал, карточки мест.
3. Кликни 🌐 в шапке → переключи на English.
   - ✅ Все строки переведены, выбор сохранён в `localStorage.wowatlas_lang = "en"`.
   - Перезагрузи — язык остался EN.
4. Переключи обратно на RU — поведение симметрично.

⚠️ Если найдёшь хардкод английский (например, «Save», «Cancel», «Search…» при RU) — это баг локализации.

## 3. Главная страница `/`

### 3.1 Welcome-модал (первый визит)
- Очисти localStorage → перезагрузи `/`.
- Должен открыться модал «Что тут можно сделать?» с 4 карточками.
- Кнопка «×» закрывает, в localStorage появляется `wowatlas_welcome_seen = "true"`.
- Перезагрузка — модал больше не появляется.

### 3.2 Header
**Десктоп:**
- Лого + подпись «Hedonist Odyssey», поиск, кнопка «🔥 Клуб», иконка 🔖 со счётчиком, 🌐, меню профиля.

**Мобила:**
- Только лого 🌌, поиск (placeholder «Поиск…»), 🌐, меню. Кнопок «Клуб» и Stash в шапке **нет** (они в bottom-nav).
- Никаких обрезанных текстов, элементы не наезжают друг на друга.

### 3.3 Sidebar (Discover)
- На десктопе слева, ширина **280px**.
- Кнопка `PanelLeftClose` сворачивает, `PanelLeft` раскрывает.
- Разделы: «Сейчас хорошо ехать», «Свайпай и открывай», «Коллекции» (Все/Киты/Сияние/Хайкинг/Каяк/История/Природа), «Скрытые места», «Нужен пропуск», «Ресурсы».
- Клик по чипу коллекции фильтрует карту → счётчик «N мест» меняется.
- На мобиле sidebar скрывается под картой как нижняя панель 40vh.

### 3.4 Карта
- Маркеры мест видны и кликабельны.
- Зум-контролы (`+ / −`) видны bottom-right и **не перекрываются** правым тулбаром на десктопе и нижней навигацией на мобиле.
- Кнопка-маркер «Тайник» на тулбаре справа: клик → показываются только сохранённые места.

### 3.5 Тулбар слоёв (CategoryFilter)
**Десктоп:** вертикальная колонка справа, кнопки solid (без blur).
**Мобила:** горизонтальная скролл-полоса сверху карты.

Прокликай каждый слой и убедись, что он визуально включается/выключается:
- 🔖 Тайник, 🏛️ ЮНЕСКО, 📜 Коренные народы, 🍽️ Рестораны, 🏔️ Рельеф, 🌑 Световое загрязнение, 🛣️ Дороги, 🔥 Туристическая нагрузка, ⚠️ Опасности, ➕ Добавить свою карту (внешняя ссылка в TG).

### 3.6 PlaceCard
- Клик по маркеру → внизу карты появляется карточка места.
- Секции: «Когда ехать» (12 месяцев), «Как добраться», «Только здесь», «Из клуба».
- Кнопка «×» (top-right) закрывает карточку, фон **solid** (без backdrop-blur).
- Кнопка «🔖 В тайник» — добавляет место, счётчик в bottom-nav/header растёт.

### 3.7 Bottom Nav (только мобила, < 768px)
- Видна 4 пункта: **Карта · Маршрут · Клуб · Тайник**.
- Активный пункт подсвечен `text-primary`.
- На «Тайник» бейдж показывает количество, если > 0.
- Footer на мобиле **скрыт**.

### 3.8 Свайп-режим (Explore)
- Из welcome-модала или Discover-панели → «Свайпай и открывай».
- Полноэкранный режим, можно листать карточки.
- Кнопка «Закрыть» возвращает на карту.

## 7. Перформанс (Network + Lighthouse)

### 7.1 Lazy loading
1. Открой DevTools → Network → фильтр **JS**, очисти.
2. Перезагрузи `/`.
3. Убедись, что **НЕ** грузятся чанки: `ClubFeed`, `TripPlanner`, `CreatorDashboard`, `ClubPost`, `CookiePolicy`.
4. Кликни «Клуб» в bottom-nav → подгружается чанк `ClubFeed-*.js`.

### 7.2 Lighthouse (Chrome DevTools → Lighthouse)
- Performance ≥ 70 (mobile), ≥ 85 (desktop).
- Accessibility ≥ 90.
- SEO ≥ 90.

### 7.3 Console
- Нет ошибок (красных).
- Нет warning'ов о повторных ререндерах (`Maximum update depth exceeded`).

## 9. Cookie & GDPR

1. Очисти cookies → перезагрузи `/`.
2. Слева снизу появляется **Cookie tab** (компактный ярлычок).
3. Раскрытие показывает текст и кнопку «Принять».
4. После Accept — `localStorage.wowatlas_cookie_consent = "accepted"`, ярлычок скрылся.
5. Ссылка на `/cookie-policy` открывает страницу политики.
